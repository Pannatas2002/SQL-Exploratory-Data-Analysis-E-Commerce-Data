# SQL-Exploratory-Data-Analysis-E-Commerce-Data

Welcome to the SQL Exploratory Data Analysis project repository!  
This project demonstrates a comprehensive data exploration and analytics workflow using a Brazilian E-Commerce Public Dataset by Olist in Kaggle, delivering actionable business insights across customer behavior, delivery operations, and revenue performance.

> This project is the first self-initiated project that I work on after I finished the course.

## 📊 Analysis Overview

### 1. 🔍 Check Data
Exploratory data profiling to understand dataset structure, identify nulls, duplicates, and inconsistencies before diving into analysis.

- Row counts & schema inspection  
- NULL value detection  
- Duplicate record checks  
- Data range & distribution overview  

---

### 2. 👥 Customer Segmentation
Segmenting customers based on purchasing behavior to enable targeted business strategies.

- Customer classification by order frequency and spending  
- High-value vs. low-engagement customer groups  
- Segment-level summary statistics  

---

### 3. 🚚 Delivery Performance
Analyzing logistics and fulfillment efficiency to identify bottlenecks and improvement areas.

- On-time vs. late delivery rates  
 

---

### 4. 💰 Revenue Analysis
Deep-dive into revenue patterns to support strategic business decisions.

- Monthly revenue trends across 2016–2018 
 

---

## 💡 Key Business Insights

- 📌 At Risk customers is 39% from all of the customers
- 📌 Last Mile delivery approximate 9 days which is 82% of total delivery time
- 📌 On-time delivery rate is 91%
